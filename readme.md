# Prism Web CSS Library

### Usage

Install dependencies:

```bash
yarn install
```

Run the watch or build script:

```bash
yarn run build:css
yarn run watch:css
```
Storybook

```
yarn storybook
```

As a test, Tokens are coming from the Supernova CSS Exporter. 
Please do not edit the `src/variables.css` file!!!!!!!
